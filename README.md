# LIST Boilerplate

#### Laravel, Inertia, Svelte, TypeScript with TailwindCSS, Flowbite.


## Install
```
cp .env .env.example
composer install
npm install
```

## Running aplication - Laravel
```
php artisan key:generate
php artisan storage:link
php artisan serve
```

## Running aplication - NPM
```
npm run dev
```

### And you should be ready to go.
